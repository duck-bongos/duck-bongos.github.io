May 2, 2023
Released first version of this website. I'm not usually one for web development, but this is my best attempt!
Summer 2022
Worked on the Machine Learning team at VideaHealth. They're a terrific organization dedicated to improving the health of millions. Got to write functions to help build an FDA-Approved medical device.
2021 - 2023
Returned to earn my M.S. in Computer Science at Stony Brook University to learn more about visual computing. Worked on my thesis, Non-Rigid Registration with Deep Learning and Conformal Harmonic Maps with Dr. Xianfeng Gu.
2017-2021
I worked as a data engineer who worked on workflows to automate data flows for financial experts. I'm most proud of implementing a document classification system. Then I moved to the Quality Automation team, my favorite time at Bloomberg. We automated checks to run on billions of data points for many different teams and financial security types. I got to author and maintain my own Python library, learned how to use the Factory software pattern to let the data tell us how it should be checked, rather than the other way around. Learned a lot about config-driven programming and empathetic programming.
Fall 2014 - Summer 2016
Worked at GE Aviation as an Information Technology Leadership Program (ITLP) intern. Worked on four different departments across four semester-long rotations: Engineering Tooling, Cloud Application, Supply Chain Analytics, Security Protocols. Learned a lot about organizational policies and how widely the word "technology" can vary!
2014
Worked at Ferno Washington, a medical device company that manufactures just about everything you can find in an ambulance. Wrote multipage excel formulas and first became interested in programming thinking "there must be a better way!"
Summer 2013
Interned at Paycor helping the sales team out where I first became interested in analytics projects.
